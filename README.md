octave-map
==========

OCTAVE Mapping Package

This package implements simple mapping functions.

  * azimuth
  * reckon
  * distance
  * departure
  * vincenty
  * vincentyDirect
  * gc2sc
  * deg2km
  * km2deg
  * deg2rad
  * rad2deg
  * geocode
  * reverse
  * npi2pi
  * zero22pi
  * referenceEllipsoid

Documentation: http://octave-mapping-package.readthedocs.org/en/latest/index.html
